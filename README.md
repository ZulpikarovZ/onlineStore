# onlineStore.
Стек:
Server - express, PostgreSQL, sequelize, jsonwebtoken и т.д.
Client - react, mobx, react-router-dom, react-bootstrap, axios и т.д.

## npm run dev - запуск сервера

## npm start - запуск клиента
